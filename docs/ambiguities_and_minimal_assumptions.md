# Ambiguities and minimal assumptions

1. The exact deduplication routine applied before the final analytical dataset was not fully recoverable as a standalone script.
2. Some platform column names vary across exports; alias resolution was added where needed.
3. Exact package versions were not fully traceable; minimum reasonable versions are declared in `requirements.txt`.
4. Several figures went through multiple visual refinements; the repository preserves the stable final versions consistent with the manuscript.
5. When multiple versions of a procedure existed, the most robust version aligned with the manuscript was retained.
